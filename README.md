# 朝会ジェネレーター (Asakai Generator)

チームの朝会で使えるトピックをランダムに生成するAndroidアプリです。

## 機能

- トピックのランダム生成（ボタン一つで朝会トピックを選出）
- カテゴリ別フィルタリング（アイスブレイク、チームビルディング、ヘルスチェック、おもしろ質問）
- トピック一覧の閲覧
- Material Design 3 / ダイナミックカラー対応

## 技術スタック

- **言語**: Kotlin
- **UI**: Jetpack Compose + Material 3
- **最小SDK**: Android 8.0 (API 26)
- **ターゲットSDK**: Android 14 (API 34)

## ビルド方法

```bash
./gradlew assembleDebug
```
